Examples + some minor 'General' info. This library is a wrapper around chart.js, so if you have questions, you often need to go looking at chart.js (stackoverflow posts) and try to map their fix to the wrapper-implementation of ng2-charts

Seems like an ok solution for simple charts

Why this library? Initially looked at swimlane/ngx-charts but that one hasn't had a release for 4 months and > 600 open Github issues. There also was no 'real' documentation, just some code examples for each chart type. # weekly downloads wasn't too bad at +-130K, but ng2-charts sits at +-180K, has had a release last month and 'only' 120-ish open Github issues
